# Age Calculator App with Kotlin

This simple mobile app, built with Kotlin, calculates a person's age in minutes and seconds. The app serves as a beginner-friendly project to understand Android app development with Kotlin and can be integrated with Android Studio for development.

## Project Structure

- `app/src/main/`
  - `java/` - Contains the Kotlin source files for the app
  - `res/` - Resource files such as layouts, strings, and drawables
  - `AndroidManifest.xml` - Configuration file for the app

## Dependencies

The Age Calculator app utilizes the following dependencies:

- Kotlin Standard Library
- AndroidX libraries
- Material Components for Android

## Development Environment

To build and run the app, you'll need:
- IntelliJ IDEA with Kotlin plugin installed
- Android SDK
- Android Virtual Device (AVD) or a physical Android device for testing

## Getting Started

To set up the development environment and run the app:

1. Clone the repository or download the source code.
2. Open the project in IntelliJ IDEA.
3. Ensure that the required SDKs and plugins are installed.
4. Connect an Android device or set up an Android Virtual Device for testing.
5. Build and run the app from IntelliJ IDEA.

## Features and Implementation

The app consists of a single screen where the user can input their date of birth. Upon submission, the app calculates and displays the user's age in minutes and seconds. This calculation is based on the current date and time.

The core logic of the age calculation can be found in the main Kotlin file `AgeCalculator.kt`. The app uses the system's date and time functions to accurately determine the age in minutes and seconds.
